# Used Vehicle Price Analysis

This repository contains a structured CRISP-DM workflow analyzing
drivers of used vehicle pricing.

## Overview

-   Dataset: 426,880 used vehicle listings
-   Objective: Identify structural drivers of price
-   Final Model: Ridge Regression
-   Performance: RMSE_log ≈ 0.897, R² ≈ 0.444

## Key Findings

-   Brand positioning strongly influences price
-   Clean titles materially increase valuation
-   Vehicle condition and recency drive pricing
-   Regional variation exists
